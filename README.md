Student Career Confusion Analyzer | Data Science Project
Technologies: Python, Pandas, NumPy, Matplotlib, Seaborn
* Designed a data-driven system to analyze student interests, academic performance, and skill preferences.
* Processed and cleaned student survey datasets using Pandas to handle missing and inconsistent values.
* Performed exploratory data analysis to identify patterns influencing career decision uncertainty.
* Built a rule-based / analytical recommendation logic to suggest suitable career paths based on data trends.
* Visualized insights using Matplotlib and Seaborn to support data-driven career guidance.
